# AI实时目标检测作业
## 个人信息
- 学号：202335020615
- 姓名：黄寿昱
- 前沿AI项目：浏览器端TensorFlow.js目标检测

## 项目核心功能
1. 前端实时调用摄像头进行AI目标检测
2. 可视化参数调整面板
3. 纯前端部署，无后端依赖

## ✅ 可迁移性
本项目仅包含 **HTML/CSS/JS** 静态文件，可直接：
- 迁移到任意GitHub/GitLab仓库
- 迁移到任意静态网站服务器（Nginx、Apache、Vercel）
- 迁移到本地电脑直接双击运行

## ✅ 可优化点（已完成基础优化）
1. **性能优化**：使用轻量AI模型 `lite_mobilenet_v2`，降低浏览器负载
2. **流畅度优化**：使用 `requestAnimationFrame` 提升检测帧率
3. **代码优化**：模块化拆分，预加载模型，减少重复请求
4. **UI优化**：响应式界面，适配手机/电脑

## ✅ 可调参数（实时修改）
1. **置信度阈值**：0.1~0.9（过滤低可信度检测结果）
2. **最大检测数量**：1~20（限制同时检测的目标数）
3. 可扩展参数：模型类型、检测类别、画面尺寸

## 技术栈
- 前沿AI框架：TensorFlow.js（浏览器端边缘计算）
- 模型：COCO-SSD 轻量目标检测
- 前端：HTML5 + CSS3 + JavaScript

## 运行方式
1. 开启摄像头权限
2. 点击「启动AI检测」
3. 滑动滑块调整参数
